<h2>Dasar Teori</h2>

<h3>Apa itu Jetpack Compose?</h3>
<p>Jetpack Compose adalah toolkit UI dari Google untuk membangun tampilan aplikasi Android secara deklaratif menggunakan Kotlin. Pendekatannya berbeda dari cara lama yang menggunakan file XML terpisah untuk desain tampilan.</p>

<table>
  <tr><th>Aspek</th><th>XML (Pendekatan Lama)</th><th>Jetpack Compose</th></tr>
  <tr><td>Cara menulis UI</td><td>File XML terpisah</td><td>Fungsi Kotlin dengan @Composable</td></tr>
  <tr><td>Pendekatan</td><td>Imperatif</td><td>Deklaratif</td></tr>
  <tr><td>Update UI</td><td>Manual lewat findViewById</td><td>Otomatis saat state berubah</td></tr>
  <tr><td>Preview</td><td>Layout Editor</td><td>Anotasi @Preview di IDE</td></tr>
</table>

<h3>Komponen yang Dipakai</h3>
<p><strong>@Composable</strong> adalah anotasi yang menandai sebuah fungsi sebagai komponen UI. Fungsi ini hanya bisa dipanggil dari fungsi @Composable lain.</p>
<p><strong>Column</strong> adalah layout yang menyusun elemen secara vertikal, mirip LinearLayout dengan orientasi vertical di pendekatan XML lama.</p>
<p><strong>Text</strong> adalah komponen untuk menampilkan teks. Parameter seperti fontSize, fontWeight, dan color diatur langsung sebagai argumen fungsi.</p>
<p><strong>Modifier</strong> digunakan untuk mengatur ukuran, padding, dan alignment komponen. Sifatnya berantai sehingga bisa disambung terus.</p>
<p><strong>Spacer</strong> adalah komponen kosong untuk memberi jarak antar elemen.</p>

import android.os.Bundle
import androidx.activity.ComponentActivity
import androidx.activity.compose.setContent
import androidx.activity.enableEdgeToEdge
import androidx.compose.foundation.layout.Arrangement
import androidx.compose.foundation.layout.Column
import androidx.compose.foundation.layout.Spacer
import androidx.compose.foundation.layout.fillMaxSize
import androidx.compose.foundation.layout.height
import androidx.compose.foundation.layout.padding
import androidx.compose.material3.MaterialTheme
import androidx.compose.material3.Scaffold
import androidx.compose.material3.Text
import androidx.compose.runtime.Composable
import androidx.compose.ui.Alignment
import androidx.compose.ui.Modifier
import androidx.compose.ui.text.font.FontWeight
import androidx.compose.ui.text.style.TextAlign
import androidx.compose.ui.tooling.preview.Preview
import androidx.compose.ui.unit.dp
import androidx.compose.ui.unit.sp
import com.farhan.helloandroid.ui.theme.HelloAndroidTheme

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContent {
            HelloAndroidTheme {
                Scaffold(modifier = Modifier.fillMaxSize()) { innerPadding ->
                    HelloScreen(modifier = Modifier.padding(innerPadding))
                }
            }
        }
    }
}

@Composable
fun HelloScreen(modifier: Modifier = Modifier) {

        horizontalAlignment = Alignment.CenterHorizontally
    ) {

            text = "Hello, Android!",

            textAlign = TextAlign.Center,
            color = MaterialTheme.colorScheme.primary
        )
        Spacer(modifier = Modifier.height(24.dp))

            text = "Mohammad Farhan Nazrul Ilhami",
            fontSize = 18.sp,
            fontWeight = FontWeight.Medium,
            textAlign = TextAlign.Center
        )

        Text(
            text = "NRP: 5025231053",
            fontSize = 14.sp,
            textAlign = TextAlign.Center,
            color = MaterialTheme.colorScheme.onSurfaceVariant
        )
        Spacer(modifier = Modifier.height(4.dp))

            text = "Pemrograman Perangkat Bergerak C",

<h2>Penjelasan Kode</h2>

<h3>MainActivity dan setContent</h3>
<p>ComponentActivity adalah kelas Activity yang direkomendasikan untuk proyek Compose. Fungsi setContent menggantikan setContentView yang biasa dipakai di pendekatan XML. Semua tampilan Compose didefinisikan di dalam blok ini.</p>
<code class="mono">setContent {
    HelloAndroidTheme {       // tema aplikasi
        Scaffold(...) {       // layout dasar
            HelloScreen(...)  // composable yang kita buat
        }
    }
}</code>

<h3>Column dan Alignment</h3>
<p>Column dengan verticalArrangement = Arrangement.Center dan horizontalAlignment = Alignment.CenterHorizontally membuat semua elemen di dalamnya berada di tengah layar, baik secara vertikal maupun horizontal.</p>
<code class="mono">Column(
    modifier = modifier.fillMaxSize().padding(32.dp),
    verticalArrangement = Arrangement.Center,
    horizontalAlignment = Alignment.CenterHorizontally
) { ... }</code>

<h3>Modifier</h3>
<p>Modifier digunakan untuk mengatur tampilan dan perilaku komponen. Sifatnya berantai sehingga beberapa pengaturan bisa digabung dalam satu baris.</p>
<code class="mono">Modifier
    .fillMaxSize()   // memenuhi seluruh layar
    .padding(32.dp)  // jarak 32dp dari tepi</code>

<h3>@Preview</h3>
<p>Dengan anotasi @Preview, Android Studio bisa merender tampilan langsung di editor tanpa perlu emulator. Sangat membantu saat mengecek perubahan desain.</p>
<code class="mono">@Preview(showBackground = true)
@Composable
fun HelloScreenPreview() {
    HelloAndroidTheme {
        HelloScreen()
    }
}</code>
